About Me

Hi, I'm Hemanth, a Data Scientist with expertise in AI/ML, skilled in building and deploying models. I'm passionate about transforming data into actionable insights using tools like TensorFlow and AWS.

I am Hemanth Vankudoth, a passionate and driven Mechanical Engineering student at IIT (BHU), Varanasi, with a keen interest in Artificial Intelligence and Machine Learning. With a strong foundation in programming languages like Python, C++, SQL, and Golang, I have developed various projects focusing on AI-driven solutions, including travel agent bots, drowsiness detection systems, and chatbots for multiple applications such as agriculture and programming assistance. I specialize in leveraging deep learning, natural language processing, and machine learning algorithms to create innovative solutions. My projects demonstrate a hands-on approach to solving real-world problems, and my work in data analysis and visualization has been recognized in competitions. Through various roles, including Under Secretary General for Creative Affairs at IIT-BHU MUN, I have also contributed to enhancing online presence and improving user experience. I am always eager to learn, collaborate, and apply my skills to new challenges.

Projects

Projects That Define My Passion and Expertise.

Real-Estate X

Developed a property search platform for Gurgaon with features including a recommender system using TFIDF and cosine similarity, a price predictor and analytics with Plotly and Seaborn. Hosted on AWS EC2 for scalable deployment.

AI-Travel Agent - (LLM's Agent)

Developed Expert Travel Agent and City Selection Agent models using Llama3 for travel planning and destination recommendations through advanced decision-making algorithms and custom tools. Additionally, created a RAG-based AI Agent that delivers contextually relevant answers (e.g., from PDFs) with approximately 90% accuracy.

Drowsiness detection

The Drowsiness Detection project uses a fine-tuned Ultralytics YOLOv5 model to detect real-time drowsiness with high accuracy. Images were annotated using labelImg, and an alarm system is activated upon detection. This system aims to enhance driver safety and workplace monitoring.

LLM Chatbots

Developed various chatbots using open-source models and APIs. The Code-Chatbot is a standout, built with Codellama to assist with programming queries. It offers effective code solutions and guidance through an intuitive Gradio interface. Also created an Image Describer using Gemini API for image accessibility and an Agri-chatbot for agriculture-related questions.

Credit card Fraud detection

Developing a credit risk model to help banks accurately assess borrowers' creditworthiness and mitigate risks associated with loan defaults. This project leverages two datasets (an internal product dataset and bureau dataset) to predict the likelihood of loan approval, with users being categorized into different priority levels.
